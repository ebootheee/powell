# Roadmap

## Now (v1)
- Core visualization: 3-year historical elevation chart with threshold lines
- Interactive forecast sliders (inflow scenario, release rate)
- Consequence panel showing what happens at projected low point
- Deploy to Azure SWA

## Next (v1.1)
- Add SNOTEL snowpack data as leading indicator overlay (if API stabilizes)
- Year-over-year comparison view (overlay multiple years)
- Bureau of Reclamation 24-Month Study projections as reference line
- Share/embed functionality

## Later (v2)
- Historical context: show every year since 1963 as faded background lines
- Inflow breakdown by tributary
- Evaporation and bank storage visualization
- Lake Mead companion view (downstream system)
- Email/push alerts when level crosses thresholds
- Download data as CSV
